The Multimodal Diagram System (MDS) uses color-coded 540px by 960px images, which are created in an external application, and then labelled using our diagram upload interface. This page includes templates and acceptable color palettes in various formats, including Powerpoint, Keynote and image editors, as well as some guidelines for diagram creation.

PPT Format
For use with Microsoft powerpoint. Build your diagram off of this blank template, then export your diagram slide as a 540x960 pixel JPG, JPEG or PNG images. If you have diagram upload size issues, lower the export quality of your JPG/JPEG images, or try PNG format.

Blank Diagram Template and Color Palette
Open the blank template in PowerPoint, then bring in your reference diagram and create a simplified diagram over it using shapes, lines, etc. Use the eyedropper tool and the color palette reference to make your diagram elements different colors. Delete your reference diagram when you are done.

Guidelines
All diagram images must be 540x960 pixels.

Elements (items with unique labels) should be color coded using contrasting colors from the color palette provided.

The background color (dead space / whitespace / blank area) for your diagram should be added as an element and labelled to non-visually indicate that a user has gone outside of the diagram.

If scale, shape or location is important for the student to understand, making sure to note this in the instructors notes and / or element description. Testing has shown users tend to focus on the audio feedback rather than spatial information when using this application unless cued.

Enlarge small elements so that users will be able to interact with them via fingers on the touchscreen. Elements under 30x30 pixels are not recommended.

A black background with lighter elements is recommended to reduce light-scatter for low vision users.

Simplify geometry as much as possible for non-visual diagrams. Currently, this system preforms well for teaching spatial relations of diagram elements (i.e. layers of the atmosphere) but is limited in its ability to tactilely present geometries.
