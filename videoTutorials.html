The collection of tutorials on this page should help students and educators get started using and developing for the MDS application.

Using the MDS Application

This video tutorial is a good starting point for beginners, for students and educators alike. This video demonstrates how to load existing diagram projects to the mobile application and also introduces piecewise and static loading.

Diagram Creation Tutorial

This video tutorial demonstrates how to create a diagram using Keynote, although the diagram creation techniques shown are applicable to your preferred program. Additionally, this tutorial demonstrates how to use the diagram upload interface to label and share your diagram.
